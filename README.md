\# Slope - Unity WebGL Game with Cheats



This is a Unity-based WebGL game called \*\*Slope\*\*, now enhanced with cheat functionality.



\## Files Included



\### Core Game Files

\- `Build/slope.json` - Build manifest

\- `Build/slope\_data.unityweb` - Game data

\- `Build/slope\_framework.unityweb` - Framework

\- `Build/slope\_memory.unityweb` - Memory

\- `Build/slope\_wasmcode.unityweb` - WASM code

\- `Build/slope\_wasmframework.unityweb` - WASM framework



\### Template Files

\- `TemplateData/UnityProgress.js` - Progress bar logic

\- `TemplateData/progressEmpty.Dark.png` - Progress bar (empty)

\- `TemplateData/progressFull.Dark.png` - Progress bar (full)

\- `TemplateData/progressLogo.Dark.png` - Progress bar (logo)

\- `TemplateData/style.css` - Styles

\- `TemplateData/unityloader41.js` - Unity loader



\### Entry Point

\- `index.html` - Main entry point



\### Cheat System

\- `cheats.js` - Cheat code injection (new file)



\## How to Run



1\. Open `index.html` in a modern browser (Chrome, Firefox, etc.).

2\. Wait for the game to load.

3\. A \*\*cheat console\*\* will appear in the bottom-right corner of the screen.



\## Cheat Console



Click the buttons in the cheat console to activate cheats:



\- \*\*Infinite Speed\*\* - Make the player move faster.

\- \*\*Infinite Score\*\* - Set the score to a high value.

\- \*\*Toggle Gravity\*\* - Disable gravity to make the player float.

\- \*\*Toggle Invincibility\*\* - Set the player's health to a high value.

\- \*\*Toggle Time Speed\*\* - Slow down or speed up the game.

\- \*\*Reset Game\*\* - Reload the page to reset the game.



\## Deployment



To deploy to GitHub Pages:



1\. Push all files (including `cheats.js`) to your GitHub repository.

2\. Go to your GitHub repository settings.

3\. Enable \*\*GitHub Pages\*\* and select the `main` branch.

4\. Your site will be live at `https://yourusername.github.io/catfoolyou/Slope`.



\## Notes



\- The cheat system uses JavaScript injection to modify runtime variables.

\- Some cheats may require the game to expose specific variables.

\- If a cheat doesn't work, check the browser console for errors.



Enjoy the game with cheats!
